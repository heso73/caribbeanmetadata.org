Famille 01
Linguistique
Décrit les langues, dialectes, registres et pratiques langagières présents dans le contenu caribéen.

Langues officielles CMS (10)

gcf — Créole guadeloupéen hat — Créole haïtien acf — Créole antillais jam — Jamaican Patois pap — Papiamentu srn — Sranan Tongo fra — Français caribéen eng — English Caribbean spa — Español caribeño nld — Nederlands Caribisch

Signaux algorithmiques

  • Code langue ISO 639-3 — identifie un cluster d'audience précis (+15 pts IVA)
  • Langue endogène caribéenne — signal de niche culturelle (+5 pts IVA)
  • Code-switching documenté — richesse linguistique caribéenne
  • Registre oral vs écrit — indexation différenciée

Exemple de fiche

"lang_code": "gcf",
"lang_endogenous": true,
"lang_codeswitching": ["gcf", "fra"],
"lang_register": "oral",
"lang_script": "Latin"

Pourquoi c'est crucial

  • Le créole guadeloupéen est parlé par 400 000 personnes — c'est un marché d'audience réel
  • Les plateformes utilisent le code langue pour cibler les recommandations géographiques
  • Sans code ISO 639-3, le contenu créolophone est invisible

Exemple de fiche

"cms_cultural_markers": [
  "gwo_ka",
  "bele",
  "carnival"
],
"cms_domain": "music",
"cms_marker_count": 3

Signaux algorithmiques

  • 1+ marqueur — connexion à une communauté d'audience (+8 pts IVA)
  • 3+ marqueurs — enrichissement du graphe de similarité (+7 pts)
  • 5+ marqueurs — signal de richesse culturelle maximale (+5 pts)

Famille 03
Narrative
Domaine culturel, titre en langue originale, description sémantique — le récit de l'œuvre en ses propres termes.

14 domaines culturels

MusiqueTradition oraleArts visuelsLittératureThéâtreGastronomieÉducationCarnavalDansePatrimoine immatérielReligionActualitésRéseaux sociauxGénéral

Signaux algorithmiques

  • Domaine culturel — catégorisation des contenus (+8 pts IVA)
  • Titre renseigné — indexation moteurs de recherche (+4 pts)
  • Titre en langue originale — recherche multilingue (+3 pts)
  • Description 50+ mots — analyse NLP algorithmique (+5 pts)

Exemple de fiche

"title": "Tambours de Guadeloupe",
"title_original": "Tanbou Gwadloup",
"cms_domain": "music",
"description": "Documentaire sur la pratique du Gwo Ka à Sainte-Anne, suivi de trois maîtres tambouyers...",
"description_lang": "gcf"

Rythmique

Tags rythmiques

Signaux algorithmiques

  • Tag rythmique identifié — signal de style pour Spotify, YouTube Music (+6 pts IVA)
  • Permet le regroupement par style sonore dans les playlists algorithmiques
  • Signal de découverte pour les algorithmes de similarité musicale

Exemple de fiche

"cms_rhythmic_tags": [
  "syncopé",

Famille 05
Géographique
Ancrage territorial précis — les 20 territoires caribéens et la diaspora mondiale.

20 territoires officiels CMS

GuadeloupeMartiniqueHaïtiJamaicaTrinidadCubaBarbadosDominican RepublicPuerto RicoArubaCuraçaoSurinameDominicaSt LuciaSt VincentGrenadaAntiguaGuyaneDiaspora caribéenneCaribbean (général)

Signaux algorithmiques

  • Territoire caribéen identifié — géolocalisation audience (+12 pts IVA)
  • Territoire spécifique (non diaspora) — niche géographique précise (+3 pts)
  • Signal direct pour les recommandations géolocalisées des plateformes

Exemple de fiche

"cms_territory": "Guadeloupe",
"cms_territory_type": "specific",
"cms_production_territory": "Guadeloupe",
"cms_diaspora": false,
"cms_region": "Antilles françaises"

Note sur la diaspora

  • La catégorie "Diaspora caribéenne" est disponible pour les œuvres sans territoire unique
  • La v3.0 prévoit un tableau de territoires pour les œuvres transnationales

IVA = Σ (signaux renseignés × poids) / 100
