Guide d'intégration

Afficher le badge CMS sur votre site

Deux lignes de code pour montrer qu'une œuvre est certifiée conforme au Caribbean Metadata Standard — territoire, langue et niveau de complétude affichés automatiquement.

Prérequis

L'œuvre doit déjà être enregistrée et certifiée dans le CMS. Si elle n'a pas encore de cms_id, consultez d'abord le validateur ou contactez contact@caribbeanmetadata.org pour l'enregistrer.

1

Trouvez le cms_id de l'œuvre

C'est l'identifiant unique au format CMS-XXX-YYYY-XXXXXXXX, visible sur la fiche de certification de l'œuvre.

2

Collez ce code à l'endroit voulu

Sur WordPress : dans un bloc "HTML personnalisé". Sur Wix/Squarespace : dans un élément "Code embed". En HTML brut : directement dans la page.

<span class="cms-badge" data-cms-id="CMS-XXX-YYYY-XXXXXXXX"></span>
<script src="https://caribbeanmetadata.org/embed/badge.js" async></script>

Remplacez CMS-XXX-YYYY-XXXXXXXX par le véritable identifiant de l'étape 1.

3

C'est tout

Le badge se charge automatiquement et affiche le territoire et le niveau de certification de l'œuvre, avec un lien vers sa fiche officielle.

Si l'identifiant est invalide ou l'œuvre introuvable, rien ne s'affiche — pas de badge cassé visible sur votre site.
