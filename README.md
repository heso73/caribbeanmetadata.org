## 🇫🇷 Français

### Présentation

Le **Caribbean Metadata Standard (CMS)** est le premier cadre unifié pour classifier, décrire et indexer les contenus audiovisuels caribéens. Développé par le **Caribwood Language Lab** (Les Abymes, Guadeloupe), il vise à mettre fin à l'invisibilité algorithmique des créateurs de la région caribéenne.

> *"La Caraïbe n'est pas un fragment du monde : elle est un monde entier."*  
> — Déclaration Fondatrice du CMS, 2026

### Les 6 familles de métadonnées

| # | Famille | Description |
|---|---------|-------------|
| 01 | **Linguistique** | Créole · Patois · Code-switching |
| 02 | **Culturelle** | Carnaval · Spiritualité · Diaspora |
| 03 | **Narrative** | Réalisme magique · Saga · Thriller |
| 04 | **Rythmique** | Tempo · Syncopé · Percussif |
| 05 | **Géographique** | Guadeloupe · Haïti · Jamaïque |
| 06 | **Socio-historique** | Créolisation · Migration · Mémoire |

## 🇬🇧 English

### Overview

The **Caribbean Metadata Standard (CMS)** is the first unified framework for classifying, describing and indexing Caribbean audiovisual content. Developed by the **Caribwood Language Lab** (Les Abymes, Guadeloupe), it aims to end the algorithmic invisibility of creators from the Caribbean region.

> *"The Caribbean is not a fragment of the world: it is a whole world."*  
> — CMS Founding Declaration, 2026

### The 6 Metadata Families

| # | Family | Description |
|---|--------|-------------|
| 01 | **Linguistic** | Creole · Patois · Code-switching |
| 02 | **Cultural** | Carnival · Spirituality · Diaspora |
| 03 | **Narrative** | Magic realism · Saga · Thriller |
| 04 | **Rhythmic** | Tempo · Syncopated · Percussive |
| 05 | **Geographic** | Guadeloupe · Haiti · Jamaica |
| 06 | **Socio-historical** | Creolization · Migration · Memory |
